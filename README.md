# 🎰 랜덤 추첨기

별도 설치 없이 브라우저에서 바로 사용할 수 있는 한국어 랜덤 추첨 웹 앱입니다.

## 데모

👉 [GitHub Pages에서 바로 사용하기](https://deantechta.github.io/dean-helloworld/)

## 주요 기능

- **대규모 참가자 지원**: 최대 5,000명 참가자 등록
- **다수 당첨자 선정**: 최소 1명 ~ 최대 1,000명 동시 추첨
- **xlsx 파일 일괄 입력**: 엑셀 파일로 참가자 명단을 한 번에 업로드
- **드럼롤 효과**: 당첨자 발표 전 긴장감을 높이는 애니메이션
- **컨페티 효과**: 당첨자 발표 시 축하 효과
- **결과 내보내기**: 추첨 결과를 CSV 파일로 저장

## 사용 방법

### 직접 입력

1. 참가자 수를 입력하고 **확인** 클릭
2. 당첨 인원 수 설정 (기본값: 1명)
3. 참가자 이름을 각 입력 칸에 입력
4. **추첨 시작** 버튼 클릭

### xlsx 파일로 일괄 입력

1. 첫 번째 열(A열)에 참가자 이름이 나열된 xlsx 파일 준비
2. **xlsx 파일 선택** 버튼으로 파일 업로드
3. 자동으로 참가자 명단이 채워진 후 추첨 진행

### 결과 내보내기

- 추첨 후 **엑셀로 내보내기** 버튼 클릭 → `draw_results_YYYYMMDD_HHMMSS.csv` 파일 저장

## 실행 방법

서버나 빌드 도구 없이 `index.html` 파일을 브라우저로 열면 바로 사용할 수 있습니다.

```bash
# 파일을 직접 열거나
open index.html

# 간단한 로컬 서버 사용 (선택)
npx serve .
```

## 기술 스택

- HTML / CSS / JavaScript (순수 바닐라, 프레임워크 없음)
- [SheetJS (xlsx)](https://sheetjs.com/) — xlsx 파일 파싱
